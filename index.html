Scritchy Scratchy Save Editor

Decode, inspect, edit, and re-encode exported saves in the browser.

Local-only tool

This editor runs entirely in your browser. Your save data is not uploaded anywhere by this page. Paste a Base64 export, decode it, edit quick fields or raw JSON, then encode it again for import back into the game.

Safer workflow Keep one untouched export as backup before importing modified saves.
Unofficial tool This is a fan-made editor. Broken or inconsistent values can produce invalid saves.
