Scritchy Scratchy Save Editor

Load a save, edit it through dedicated sections, then generate a fresh import string locally in your browser.

Single-page editor, less raw JSON pain

This version is built around how the save actually behaves: currencies, progression, tickets, upgrades, cosmetics, achievements, dialogues, and advanced JSON.

It targets the known save structure for game version 1.0.30d.

Target game version: 1.0.30d
1. Export in game The game copies your current save to clipboard.
2. Load it here Paste the export string and decode it.
3. Edit by section Use dedicated pages for the save parts instead of raw JSON lists.
4. Import in game Generate a new string, copy it, then hold Import for 3 seconds.
