Scritchy Scratchy Save Editor

Paste an exported save, edit the values you want, then generate a new import string locally in your browser.

Local-only tool

This editor runs entirely in your browser. Your save data is not uploaded anywhere by this page.

Use one untouched export as backup before importing any edited save.

1. Export in game The game copies your current save to clipboard.
2. Edit it here Paste the export, load it, and change the values you want.
3. Import in game Copy the generated import string, then hold Import for 3 seconds.
